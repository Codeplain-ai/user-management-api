# Health Check Endpoint Conformance Tests

This directory contains conformance tests for the `/health_check` endpoint of the API.

## Overview

These tests verify that the health check endpoint meets the functional requirements by making HTTP requests to the running API and validating the responses.

## Prerequisites

- Node.js and npm installed
- The API must be running and accessible at `http://localhost:8000`
- Dependencies installed via `npm install`
